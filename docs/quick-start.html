Quick Start Guide

Get up and running with the 4Culture Grant SDK in minutes.

Step 1: Installation

Install the SDK and its dependencies:

# Clone the repository
git clone https://github.com/your-org/grants-sdk.git
cd grants-sdk

# Install Python dependencies
pip install -r requirements.txt

# Install Playwright browsers (for browser automation)
playwright install chromium

Step 2: Configuration

Set up your credentials and API keys:

# Set environment variables
export 4CULTURE_USERNAME="your_username"
export 4CULTURE_PASSWORD="your_password"
export OPENAI_API_KEY="your_openai_api_key"  # Optional, for narrative generation

💡 Tip: You can also pass credentials directly via command-line arguments.

Step 3: Verify Installation

Test that the SDK is installed correctly:

# Check CLI is available
python3 -m agents.grant_scraper.sdk_cli --help

# View available command groups
python3 -m agents.grant_scraper.sdk_cli auth --help
python3 -m agents.grant_scraper.sdk_cli data --help

Step 4: Your First Command

Extract data from grant documents:

# Extract data from documents
python3 -m agents.grant_scraper.sdk_cli data extract \
  --grant-name "Moving Stories: Short-Form Graphic Novels" \
  --docs-folder docs \
  --output extracted_data.json

This command will:

  • Search for relevant documents in the docs folder
  • Extract grant information, project details, and contact information
  • Save the extracted data to extracted_data.json

Step 5: Login to 4Culture Portal

Authenticate with the 4Culture portal:

# Interactive login (recommended for first time)
python3 -m agents.grant_scraper.sdk_cli auth login

# Or non-interactive login
python3 -m agents.grant_scraper.sdk_cli auth login \
  --username "your_username" \
  --password "your_password" \
  --non-interactive

⚠️ Note: Browser automation requires MCP tools. The SDK will guide you through setup on first use.

Step 6: Run a Complete Workflow

Automate the entire grant application process:

# Run full workflow (dry-run mode)
python3 -m agents.grant_scraper.sdk_cli workflow full \
  --grant-name "Your Grant Name" \
  --docs-folder docs \
  --use-narratives \
  --openai-api-key $OPENAI_API_KEY \
  --dry-run

This workflow will:

  1. Extract data from documents
  2. Standardize the data format
  3. Map data to form fields
  4. Generate narratives (if enabled)
  5. Fill out the application form
  6. Save the draft

Common Issues

❌ "MCP tools not available"

Solution: The SDK will automatically set up Playwright tools. If you see this error, run:

python3 -m agents.grant_scraper.mcp_setup_cli configure

❌ "OpenAI API key required"

Solution: Set the API key as an environment variable or pass it via --openai-api-key flag.

❌ "No documents found"

Solution: Ensure your documents are in the docs folder and contain the grant name in their content.
