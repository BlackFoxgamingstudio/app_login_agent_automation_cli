Getting Started

1

Install the SDK

# Clone the repository
git clone https://github.com/your-org/grants-sdk.git
cd grants-sdk

# Install dependencies
pip install -r requirements.txt
2

Set Up Credentials

# Set environment variables
export 4CULTURE_USERNAME="your_username"
export 4CULTURE_PASSWORD="your_password"
export OPENAI_API_KEY="your_api_key"  # Optional
3

Run Your First Command

# Extract data from documents
python3 -m agents.grant_scraper.sdk_cli data extract \
  --grant-name "Your Grant Name" \
  --docs-folder docs \
  --output extracted_data.json
4

Explore the CLI

# View all available commands
python3 -m agents.grant_scraper.sdk_cli --help

# View command groups
python3 -m agents.grant_scraper.sdk_cli auth --help
python3 -m agents.grant_scraper.sdk_cli data --help
python3 -m agents.grant_scraper.sdk_cli workflow --help

Examples

Example 1: Extract Data from Documents

python3 -m agents.grant_scraper.sdk_cli data extract \
  --grant-name "Moving Stories: Short-Form Graphic Novels" \
  --docs-folder docs \
  --output extracted_data.json

Example 2: Generate Project Narrative

python3 -m agents.grant_scraper.sdk_cli narrative generate \
  --grant-name "Your Grant Name" \
  --narrative-type project_description \
  --openai-api-key $OPENAI_API_KEY \
  --output narrative.txt

Example 3: Run Complete Workflow

python3 -m agents.grant_scraper.sdk_cli workflow full \
  --grant-name "Your Grant Name" \
  --docs-folder docs \
  --use-narratives \
  --openai-api-key $OPENAI_API_KEY

Example 4: Manage Grant Database

# Add grant to database
python3 -m agents.grant_scraper.sdk_cli database add \
  --name "Grant Name" \
  --deadline "2026-01-13" \
  --status planned \
  --priority 5

# List all grants
python3 -m agents.grant_scraper.sdk_cli database list

# Generate dashboard
python3 -m agents.grant_scraper.sdk_cli report dashboard

API Reference

Command Groups

auth Authentication commands (login, logout)
data Data extraction and validation (extract, validate, standardize, identify-docs)
mapping Field mapping (map, validate-map, create-instructions)
narrative Narrative generation (generate, generate-all)
browser Browser automation (navigate, snapshot, find, click, type, select, screenshot, wait)
form Form automation (start, extract-structure, fill, fill-all, validate, save)
workflow Workflow orchestration (full, custom)
database Database management (add, update, get, list, timeline)
report Reporting (generate, dashboard)
