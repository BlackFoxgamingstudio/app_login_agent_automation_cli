# Design Preferences

Select the visual style and themes for your Google Site. Google Sites offers a few base themes that we can customize with your colors and fonts.

## Theme Selection

Please choose **ONE** of the following Google Sites base themes (type 'x' inside the brackets):
- [ ] Simple (Clean, minimal, default)
- [ ] Aristotle (Modern, serif headings)
- [ ] Diplomat (Heavy, traditional block style)
- [ ] Vision (Bold header block)
- [ ] Level (Color ribbon header)
- [ ] Impression (Elegant, soft)

## Color Palette

Set the primary color for your site theme. This will affect buttons, accents, and header overlays depending on the theme chosen above.

- **Primary Brand Color (Hex Code):** [#000000]
